# search_wechat_key
## 本程序用来搜索微信数据库的密钥key信息   
  
**不要用于非法用途！**  
**不要用于非法用途！**  
**不要用于非法用途！**  
  
原理是打开微信进程句柄，遍历微信内存空间。   
找到关键字，比如android或者iphone字样，返回这个地址。   
从这个地址开始往前逐步搜，每次读32字节，然后判断读取的是不是微信sqlite的数据库密钥。   
密钥找到后，可以为后续读取微信聊天记录做准备。
